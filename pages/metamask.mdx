# Add ApeChain to Metamask
## Using the ApeChain Hub Page
1. Navigate to our [Hub Page](https://curtis.hub.caldera.xyz/) 
2. Click the 
**Connect** button in the *Your Wallet* section, and you will be prompted to switch to the ApeChain Network

![hubpage](../images/hub.png)

## Manually via Metamask
You can add the network manually via the following details:

### Testnet
- Network Name: Curtis
- RPC URL: https://curtis.rpc.caldera.xyz/http
- Chain ID: 33111
