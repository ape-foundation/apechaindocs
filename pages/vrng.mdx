# Verified Random Number Generator (vRNG)

ApeChain's vRNG is powered by **Supra dVRF**, a distributed and cryptographically verifiable randomness service. It enables smart contracts to request tamper-proof random numbers on-chain — suitable for NFT minting, gaming, lotteries, and any use case where provable fairness matters.

Contract addresses for ApeChain Mainnet and Testnet are listed on the [Supra dVRF Available Networks ↗](https://docs.supra.com/dvrf/learn-supra-dvrf/networks) page.

## Getting Started

Full integration guides, gas configuration, and code examples are available in the [Supra dVRF documentation ↗](https://docs.supra.com/dvrf/build-third-party-evm-networks/getting-started).
